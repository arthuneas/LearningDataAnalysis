## Visão do Repositório

A análise de dados é a ferramenta que nos permite remover o "achismo" das discussões sociais. Este repositório foca na aplicação do método científico sobre bases de dados públicas (como os Microdados do ENEM), utilizando computação de alto desempenho para identificar padrões de desigualdade e configurações estruturais da sociedade.

## Metodologia da Análise 

Para garantir o rigor das observações, o fluxo de trabalho é dividido em três pilares:

1.  **Manipulação com Pandas:** Limpeza de dados brutos, tratamento de valores nulos e engenharia de atributos (*feature engineering*) para cruzar variáveis socioeconômicas.
2.  **Cálculo Científico com SciPy:** Aplicação de testes estatísticos, correlações e funções matemáticas avançadas para validar se as observações possuem significância estatística.
3.  **Visualização com Matplotlib:** Geração de gráficos customizados (histogramas, boxplots e dispersão) que facilitam a interpretação visual dos dados complexos.

## Estudo de Caso: Microdados do ENEM 

Utilizando esse conjunto de ferramentas, analisei a relação entre a escolaridade dos pais e o desempenho dos candidatos. 

A análise demonstra que a pontuação média sobe exponencialmente conforme o nível de instrução dos responsáveis (especialmente em famílias com pós-graduação), evidenciando o abismo educacional e a reprodução de capital cultural descrita na sociologia, agora comprovada por dados.
