# LearningDataAnalysis
_aprendendo sobre analise de dados e sua aplicações práticas!_

Ultilizando a biblioteca Pandas, Matplotlib e outros para fazer analises pontuais e averiguar sobre observações sociais.

Por exemplo, olhando dados públicos sobre o ENEM, podemos inferir que a pontuação mais altas derivam de participantes com pais pós graduados. 

Logo, a analise de dados faz com que observamos a configuração social e como podemos mudá-la.
